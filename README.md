# Online Retail Data Analysis (E-Commerce Project)

## Project Overview

### Goal
Провести анализ данных интернет-магазина на основе набора **Online Retail (UCI / Kaggle)**  
за период с **1 декабря 2010 по 9 декабря 2011 года**, чтобы определить:
- какие товары и страны приносят наибольшую выручку;
- когда происходят пики продаж;
- как ведут себя клиенты (частота, давность и сумма покупок);
- кто формирует основную часть дохода.

---

### Tools and Libraries
- Python 3.x  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

---

### Project Stages
| Stage | Description |
|--------|--------------|
| 1. Data Overview | Первичный осмотр структуры и пропусков |
| 2. Data Cleaning | Очистка данных, создание поля Revenue |
| 3. Exploratory Data Analysis (EDA) | Анализ товаров, стран, сезонности и клиентов |
| 4. RFM Analysis | Сегментация клиентов по Recency, Frequency, Monetary |
| 5. Final Insights | Итоговые выводы и рекомендации |

---
